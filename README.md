## Swift Seminar2017

[Swift Seminar 2017](http://code0-sh.github.io/SwiftSeminar2017/)のソースコードです。

ビルドシステムとしてGulp.jsを使用しています。

##### 必要なもの

- Node.js

- npm

##### 使い方

```bash
$ npm install -g gulp
```

```bash
$ npm install
```

```bash
$ gulp
```
